## рҹ“– Overview

SkyVista is a modern weather application that enables users to search for any city and instantly retrieve real-time weather information. Built with **React**, **Vite**, and **Tailwind CSS**, the application focuses on delivering an intuitive user experience with responsive design and clean visuals.

Whether you're planning a trip, checking today's weather, or simply curious about the climate in another city, SkyVista provides accurate and reliable weather data within seconds.
